# 서버 모니터링

등록된 서버에 SSH로 접속하여 시스템 상태와 Docker 컨테이너 정보를 실시간으로 모니터링합니다.

## 사용 방법

### 1. 서버 선택
- 상단 드롭다운에서 모니터링할 서버를 선택합니다.
- **서버 새로고침** 버튼으로 서버 목록을 갱신할 수 있습니다.

### 2. 상태 조회
- **조회** 버튼을 클릭하면 SSH로 접속하여 서버 상태를 가져옵니다.
- 자동 새로고침: **자동(10s)** 체크박스를 활성화하면 10초 간격으로 자동 갱신됩니다.

### 3. 서버 상태
다음 시스템 정보가 게이지 바와 함께 표시됩니다:
- **CPU 사용률** (%)
- **메모리 사용률** (사용 중 / 전체)
- **디스크 사용률** (사용 중 / 전체)
- Hostname, Uptime, Load Average

색상 기준:
- 초록: 60% 미만
- 노랑: 60~80%
- 빨강: 80% 이상

### 4. Docker 컨테이너
서버에 Docker가 설치된 경우 컨테이너 목록이 테이블로 표시됩니다:
- 컨테이너 ID, 이름, 이미지
- 실행 상태, CPU/메모리 사용량
- 네트워크/블록 I/O

### 5. 컨테이너 관리
- **트래킹**: 체크박스로 대시보드에 표시할 컨테이너를 선택합니다.
- **로그**: 컨테이너의 최근 로그를 조회합니다 (최근 200줄).

## 사전 설정
- **서버 관리** 탭에서 서버를 먼저 등록해야 합니다.
