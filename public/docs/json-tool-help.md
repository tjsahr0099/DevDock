# JSON 도구

JSON 데이터를 포맷팅, 압축, 검증할 수 있는 편집 도구입니다.

## 주요 기능

### 포맷팅 / 압축
- **포맷팅**: 들여쓰기를 적용하여 JSON을 읽기 쉽게 정리합니다.
- **압축**: 공백과 줄바꿈을 제거하여 한 줄로 압축합니다.
- **들여쓰기**: 2 spaces / 4 spaces / Tab 중 선택할 수 있습니다.

### 검증
- **검증** 버튼으로 JSON 유효성을 확인합니다.
- 에러 발생 시 줄 번호와 위치가 에디터에 마커로 표시됩니다.
- 실시간으로 구문 오류를 감지하여 상단에 경고를 표시합니다.

### 편집기
- 좌측: JSON 입력 에디터 (구문 강조, 줄번호)
- 우측: 변환 결과 에디터 (읽기 전용)
- 두 에디터 모두 JSON 전용 모드로 동작합니다.

### 도구 버튼
- **붙여넣기**: 클립보드 내용을 입력 에디터에 붙여넣습니다.
- **샘플**: 예제 JSON 데이터를 불러옵니다.
- **지우기**: 입력/결과 에디터를 모두 초기화합니다.
- **복사**: 결과 에디터의 내용을 클립보드에 복사합니다.
- **교환**: 좌우 에디터 내용을 교환합니다.

## 상태 표시줄
하단에 JSON 유효성, 문자 수, 줄 수가 표시됩니다.
