# 마크다운 뷰어

마크다운(`.md`) 파일을 탐색하고 미리보기 및 PDF로 변환할 수 있는 뷰어입니다.

## 사용 방법

### 1. 폴더 선택
- **찾아보기** 버튼으로 마크다운 파일이 있는 폴더를 선택합니다.
- 경로를 직접 입력 후 Enter를 눌러도 됩니다.
- 새로고침 버튼으로 파일 목록을 갱신할 수 있습니다.

### 2. 파일 선택
- 좌측 트리에서 `.md` / `.markdown` 파일을 클릭하면 우측에 미리보기가 표시됩니다.
- 폴더 구조가 그대로 반영되어 하위 폴더 탐색이 가능합니다.

### 3. 미리보기
- GFM(GitHub Flavored Markdown) 문법을 지원합니다.
  - 테이블, 체크리스트, 취소선 등
- 코드 블록에 구문 강조(syntax highlighting)가 적용됩니다.

### 4. PDF 변환
- **PDF 저장** 버튼을 클릭하면 현재 마크다운 파일을 PDF로 변환합니다.
- 저장 위치를 선택하는 다이얼로그가 표시됩니다.
- 한글 폰트가 적용된 깔끔한 PDF가 생성됩니다.

### 5. 파일 열기
- **폴더 열기** 버튼으로 선택한 파일의 위치를 파일 탐색기에서 엽니다.

## 참고
- PDF 변환에는 Chrome 또는 Edge 브라우저가 필요합니다 (Windows에서는 Edge가 기본 설치되어 있습니다).
- 좌우 패널의 크기는 구분선을 드래그하여 조절할 수 있습니다.
