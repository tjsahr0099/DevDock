# PlantUML 뷰어

PlantUML 파일을 탐색하고 다이어그램을 SVG로 렌더링하여 미리볼 수 있는 뷰어입니다.

## 사용 방법

### 1. 폴더 선택
- **찾아보기** 버튼으로 PlantUML 파일이 있는 폴더를 선택합니다.
- 지원 확장자: `.puml`, `.plantuml`, `.pu`, `.wsd`

### 2. 파일 선택
- 좌측 트리에서 파일을 클릭하면 PlantUML 서버를 통해 SVG 다이어그램이 렌더링됩니다.
- 렌더링 중에는 로딩 스피너가 표시됩니다.

### 3. 다이어그램 보기
- 렌더링된 SVG 다이어그램이 우측 패널에 표시됩니다.
- **확대/축소**:
  - `+` / `-` 버튼으로 줌 레벨 조절
  - `Ctrl + 마우스 휠`로 확대/축소
  - 퍼센트 버튼 클릭으로 100%로 리셋

### 4. 소스 코드 보기
- 하단의 **PlantUML 소스 코드** 섹션을 펼치면 원본 PUML 코드를 확인할 수 있습니다.

## 참고
- PlantUML 서버에 HTTP 요청을 보내 SVG를 생성하므로 **인터넷 연결이 필요**합니다.
- 좌우 패널의 크기는 구분선을 드래그하여 조절할 수 있습니다.
