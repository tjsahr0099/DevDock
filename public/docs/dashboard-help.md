# 대시보드

등록된 모든 서버의 상태를 카드 형태로 한눈에 모니터링할 수 있는 화면입니다.

## 주요 기능

### 서버 카드
- 각 서버의 **CPU / 메모리 / 디스크** 사용량을 게이지 바로 표시합니다.
- 서버 연결 상태를 색상으로 구분합니다 (정상: 초록, 실패: 빨강).
- 트래킹 중인 Docker 컨테이너의 상태가 카드 하단에 뱃지로 표시됩니다.

### 자동 새로고침
- **자동** 체크박스를 활성화하면 15초 간격으로 모든 서버 상태를 자동 갱신합니다.
- **새로고침** 버튼으로 수동 갱신도 가능합니다.

### 필터
- **정상 / 실패** 필터: 서버 연결 상태별 카드 표시/숨김
- **실행 / 중지** 필터: 컨테이너 실행 상태별 필터링

### 이벤트 로그
- 하단의 **이벤트 로그** 패널을 펼치면 서버 상태 변화 이력을 확인할 수 있습니다.
- 연결 실패, 복구, 임계치 초과 등의 이벤트가 시간순으로 기록됩니다.

## 사전 설정

- **서버 관리** 탭에서 서버를 먼저 등록해야 합니다.
- **서버 모니터링** 탭에서 컨테이너 트래킹을 설정하면 대시보드 카드에 표시됩니다.
