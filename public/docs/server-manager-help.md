# 서버 관리

SSH 서버 정보를 등록, 수정, 삭제하고 SSH 터미널로 접속할 수 있는 관리 도구입니다.

## 주요 기능

### 서버 목록
- 등록된 서버들이 테이블 형태로 표시됩니다.
- 서버를 클릭하면 우측 폼에 상세 정보가 로드됩니다.

### 서버 등록/수정/삭제
- **이름**: 서버 식별용 이름
- **Host (IP)**: 서버 주소
- **Port**: SSH 포트 (기본값: 22)
- **Username / Password**: SSH 접속 계정
- **설명**: 서버에 대한 메모

버튼:
- **추가**: 새 서버를 등록합니다.
- **수정**: 선택한 서버 정보를 업데이트합니다.
- **삭제**: 선택한 서버를 제거합니다.
- **초기화**: 입력 폼을 초기화합니다.

### SSH 연결
- **연결 테스트**: 서버에 SSH 접속이 가능한지 확인합니다.
- **SSH 연결**: 외부 터미널 프로그램으로 SSH 세션을 시작합니다.
  - Windows: Windows Terminal → PuTTY → cmd 순으로 시도
  - macOS: Terminal.app
  - Linux: 기본 터미널 에뮬레이터

### PuTTY 세션 가져오기
- **PuTTY 가져오기** 버튼을 클릭하면 Windows에 등록된 PuTTY 세션 목록이 표시됩니다.
- 체크박스로 원하는 세션을 선택하여 서버 목록에 일괄 추가할 수 있습니다.
- Windows 전용 기능입니다.

## 참고
- 서버 정보는 `data/servers.json` 파일에 저장됩니다.
- 등록한 서버는 **서버 모니터링**, **대시보드** 탭에서 공유됩니다.
