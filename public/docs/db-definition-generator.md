# DB 정의서

MySQL/MariaDB 데이터베이스에 접속하여 테이블 정의서 및 데이터 사전을 Excel 파일로 자동 생성합니다.

## 사용 방법

### 1. DB 연결 정보 입력
- **Host**: 데이터베이스 서버 주소 (기본값: localhost)
- **Port**: 포트 번호 (기본값: 3306)
- **Database**: 대상 데이터베이스 이름
- **Username / Password**: 접속 계정 정보
- **연결 테스트** 버튼으로 접속 가능 여부를 확인합니다.

### 2. 추가 정보
- **작성자**: Excel 문서에 기재될 작성자 이름
- **DB User**: Excel 문서에 기재될 DB 사용자명

### 3. 출력 설정
- **찾아보기** 버튼으로 Excel 파일이 저장될 폴더를 선택합니다.

### 4. 생성
- **정의서 생성**: 테이블별 컬럼 정보, 인덱스, 코멘트 등을 포함한 정의서 Excel 생성
- **데이터 사전**: 전체 테이블/컬럼을 한 시트에 정리한 데이터 사전 Excel 생성

## 출력 내용

### 테이블 정의서
각 테이블별로 다음 정보를 포함합니다:
- 컬럼명, 데이터 타입, Nullable 여부
- PK/FK 여부
- 기본값, 코멘트
- 인덱스 정보

### 데이터 사전
- 전체 테이블 목록과 컬럼 정보를 통합 정리

## 참고
- 입력한 연결 정보와 설정은 자동으로 저장되어 다음 실행 시 복원됩니다.
- 진행률이 프로그레스 바로 표시됩니다.
